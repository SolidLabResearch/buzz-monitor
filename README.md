 # Buzz Monitor

The repository contains scripts employed for the evaluation and experiment for the [Streaming Query Hive](https://github.com/SolidLabResearch/streaming-query-hive).

## Getting Started

### Prerequisites

- Node.js (>=16.0.0)
- npm

### Installation

```bash
npm install
```

### Development

Build the project:
```bash
npm run build
```

Run in development mode with file watching:
```bash
npm run dev
```

Start the application:
```bash
npm start
```

### Testing

Run all tests:
```bash
npm test
```

Run tests in watch mode:
```bash
npm run test:watch
```

Run tests with coverage:
```bash
npm run test:coverage
```

### Linting

Check code style:
```bash
npm run lint
```

Fix linting issues automatically:
```bash
npm run lint:fix
```

### Cleaning

Clean build artifacts:
```bash
npm run clean
```

## Scripts Reference

| Script | Description |
|--------|-------------|
| `npm run build` | Compile TypeScript to JavaScript |
| `npm run build:watch` | Compile with file watching |
| `npm test` | Run Jest tests |
| `npm run test:watch` | Run tests in watch mode |
| `npm run test:coverage` | Run tests with coverage |
| `npm run lint` | Run ESLint |
| `npm run lint:fix` | Fix ESLint issues |
| `npm run clean` | Remove build artifacts |
| `npm run dev` | Development mode with watching |
| `npm start` | Start the application |
